# Дверной Достык

Мини-прототип системы автоматизации склад-магазина дверей.

## Стек
- **Backend:** Python 3.12 + FastAPI + SQLAlchemy + PostgreSQL 16
- **Frontend:** Vue.js 3 + Vite + Axios

## Запуск backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Запуск frontend
```bash
cd frontend
npm install
npm run dev
```
